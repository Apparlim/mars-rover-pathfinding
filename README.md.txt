# Mars Rover Pathfinding Simulator

A visually stunning, interactive AI pathfinding simulator that demonstrates advanced search algorithms for Mars rover navigation.



## Features
- **Multiple Algorithms**: A*, Dijkstra, Greedy, Bidirectional search
- **Dynamic Terrain**: Craters, canyons, sand dunes, rocky terrain
- **Real-time Visualization**: Watch the algorithm explore and find paths
- **Interactive Controls**: Click to place start/goal positions
- **Performance Metrics**: Path cost, nodes explored, efficiency score
- **Smooth Animations**: Rover follows the calculated path

##Â  Technologies

## How to Use
1. Select a terrain type
2. Click "Generate Terrain"
3. Click on the map to set start (green)
4. Click again to set goal (red)
5. Choose an algorithm
6. Click "Find Path"
7. Watch the magic happen!

##Â  Algorithms Explained

### A* Search
Optimal pathfinding using heuristic + actual cost

### Dijkstra's Algorithm
Guaranteed shortest path without heuristic

### Greedy Best-First
Fast but not always optimal

### Bidirectional A*
Searches from both start and goal simultaneously
